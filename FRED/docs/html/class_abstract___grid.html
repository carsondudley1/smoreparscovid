Detailed Description

This class represents an abstract grid of patches.

The grid is a layer of patches that extends throughout the area in which the simulation takes place. There is typically only one of each specific layer during a simulation.

This class is inherited by Neighborhood_Layer, Regional_Layer, and Visualization_Layer.
