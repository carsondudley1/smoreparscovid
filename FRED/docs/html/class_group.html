Detailed Description

This class represents a group in which agents in the simulation can interact with each other.

Groups can take different forms; some examples are schools, households, workplaces, and social networks. It is through these groups that infection spreads. Groups can be identified by an ID or label, which are created as the program runs, or their Synthetic Population (SP) ID, which is predefined and completely unique. Groups track data on the members of the group and the spread of infection within the group.

This class is inherited by Network and Place.
