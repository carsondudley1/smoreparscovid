Detailed Description

This class represents an abstract patch in a grid.

A patch is an area of the simulation, with varying sizes dependent on the patch size. A patch acts as a single cell in it's corresponding grid / layer.

This class is inherited by Neighborhood_Patch, Regional_Patch, and Visualization_Patch.
