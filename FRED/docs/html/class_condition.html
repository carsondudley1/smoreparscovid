Detailed Description

This class represents a condition that a person may have, usually an infectious condition.

Every condition has an associated Epidemic, Natural_History, Transmission, and Transmission_Network. Conditions can be divided into different condition states, which represent the state of the condition that the host is in. Many Condition methods are simply calls to the Epidemic model's method counterparts.
