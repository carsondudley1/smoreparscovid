Detailed Description

This class represents the natural history of an infection throughout the simulation.

Natural_History is associated with a specific Condition, that condition's Epidemic model, and a State_Space model. A natural history tracks data on the course an infection takes during its spread thruoghout a population. Specifically, this class contains data corresponding to different states in the associated condition, as well as the way that condition spreads.
