Detailed Description

An admin division represents a division in the simulation of varying complexity.

An admin division is identified by an admin code, and can contain households as well as track population sizes within the division. A division can be broken down into subdivisions, and can, in turn, have a higher division.

This class is inherited by Block_Group, Census_Tract, County, and State.
